# TeamArray-Front
프론트 작업

## 개인적으로 작업할때
1. github에서 최신 코드를 받는다
2. 받은 파일을 vscode로 연다
3. `yarn` 을 쳐서 모듈을 받는다
4. `yarn start`를 쳐서 시작한다 
5. 코드 작업을 한다
6. github에 올린다

## 백엔드에 보내면 처리해 주는것들
* 로그인 => /api/login
* 회원가입 => /api/signup
* 게시글 등록
* 게시글 삭제
* 게시글 편집
* 게시글 읽기
* 게시글 목록
* 댓글 등록
* 댓글 삭제
* 댓글 편집
* 댓글 목록
